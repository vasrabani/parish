***** You need to place your repository in a folder called "parish" *****

Step 1 - Download code "parish_inspection-main"
Step 2 - go to Command prompt and go to root folder 
	- parish
Step 3 - Create virtual environment using the command:
	>
	python -m venv env

	This will create the folder /env

Step 4 - Activate the server, running the command:

	>
	env/Scripts/activate

	The path will now have the prefix (env) so you have the virtual dev env

Step 5 - Install Django - using the following command
	>
	pip install django

Step 6 - Run the following command

	>
	pip install -r requirements.txt

	Note: This is the step that imports all the library modules and all dependencies
	
	pip install django-crispy-forms
	pip install crispy-bootstrap5
	pip install pillow


Step 6 - move to manage.py location

	>
	cd parish_inspection

	Note your path should be

	.../parish/parish_inspection-main/parish_inspection
	
	There will be a file manage.py	

	.../parish/parish_inspection-main/parish_inspection/manage.py

	This is the location you need to be

Step 7

	Run Migrate : using command to create all user/system tables
	>
	python manage.py migrate

Step 8
	Create superuser using command:
	>
	python manage.py createsuperuser

	Note: add your admin and password

Step 9 - Seed the Data run the Shell command

	>
	python manage.py shell

	Note: The shell command allows you to use a space to do some data operations

Step 10 SHELL - Using the >>> prmompt type in the command then press <Enter>

	>
